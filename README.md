# Fish Configuration

## Abbreviations

### Git

-   `g` for `git`
-   `ga` for `git add`
-   `gc` for `git commit -m`
-   `gco` for `git checkout`
-   `gd` for `git diff`
-   `gl` for `git pull`
-   `gp` for `git push`
-   `gm` for `git merge`
-   `gs` for `git status`
-   `gb` for `git branch`


### Bundler

-   `b` for `bundle`
-   `bx` for `bundle exec`
